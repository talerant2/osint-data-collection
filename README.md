# osint-data-collection

# Сбор данных по открытым источникам (OSINT)

Учебный проект по автоматическому сбору данных из соцсетей, Telegram и открытых API.

## Что сделано
- Написан скрипт на Python, который собирает данные из Twitter, Telegram и открытых API.
- Проведена очистка и анализ данных.
- Подготовлен отчёт с выявленными закономерностями.

## Использованные технологии
- Python, Requests, BeautifulSoup, API

## Как запустить
1. Установи зависимости: `pip install -r requirements.txt`
2. Добавь свои ключи API в файл `.env`
3. Запусти: `python main.py`

## Результаты
Собрано более 10 000 записей, подготовлен аналитический отчёт.
